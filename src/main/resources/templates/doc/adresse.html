Adresseopslag

Find adresse ud fra id

GET /service/search/adresse/{id}

Parametre

Navn Beskrivelse
id Adressens id. Denne er unik, tildelt af systemet, og modtages i data ved søgning.

Eksempler


    
Find alle adresser i Grønland, og output data i xml-format

    
/services/search/adresse?land=gl&format=xml


    
Find alle adresser på 2. etage

    
/services/search/adresse?etage=2


    
Find alle adresser i kommuner som begynder med "a", på vejkoder som ender på 4, i postnumre som begynder med 6

    
/services/search/adresse?kommune=a*&vej=*4&post=6*
