Lokalitetopslag

Find Lokalitet ud fra id

GET /service/search/lokalitet/{id}

Parametre

Navn Beskrivelse
id Lokalitetens id. Denne er unik, tildelt af systemet, og modtages i data ved søgning.

Eksempler


    
Find alle lokaliteter i Grønland, og output data i xml-format

    
/services/search/lokalitet?land=gl&format=xml


    
Find alle lokaliteter ved navn "Nuuk"

    
/services/search/lokalitet?lokalitet=Nuuk


    
Find alle lokaliteter i Grønland som slutter på "ut"

    
/services/search/lokalitet?land=gl&lokalitet=*ut
