Adgangsadresseopslag

Find adgangsadresse ud fra id

GET /service/search/adgangsadresse/{id}

Parametre

Navn Beskrivelse
id Adgangsadressens id. Denne er unik, tildelt af systemet, og modtages i data ved søgning.

Find alle adgangsadresser i Grønland, og output data i xml-format

    
/services/search/adgangsadresse?land=gl&format=xml


    
Find alle adgangsadresser i kommuner som begynder med "a", på vejkoder som ender på 4, i postnumre som begynder med 6

    
/services/search/adgangsadresse?kommune=a*&vej=*4&post=6*
