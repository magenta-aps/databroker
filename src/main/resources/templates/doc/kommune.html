Kommuneopslag

Find kommune ud fra kommunekode

GET /service/search/kommune/{kommunekode}

Parametre

Navn Beskrivelse
kode Kommunens unikke kode

Eksempler

Hent data på kommunen med kode 751 (Århus)

/services/search/kommune/751

Kommunesøgning

Søg efter kommuner, afgrænset af angivne parametre

GET /service/search/kommune

Parametre

Navn Beskrivelse

Eksempler

Find alle kommuner i Grønland, og output data i xml-format

/services/search/kommune?land=gl&format=xml

Find alle kommuner hvis kommunekode begynder med tallet 5

/services/search/kommune?kommune=5*

Find alle kommuner i Danmark hvis navn ender på "borg"

/services/search/kommune?land=dk&kommune=*borg
